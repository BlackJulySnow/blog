# Hello World! 👋

这是我的 Hugo 博客的第一篇文章。

## 为什么选择 Hugo + Hextra？

### Hugo 的优势

Hugo 是目前世界上最快的静态网站生成器：

- ⚡️ **极速构建** - 每秒生成数千页面
- 📝 **Markdown 原生支持** - 专注写作
- 🎨 **主题丰富** - 300+ 主题可选
- 🚀 **单二进制文件** - 单文件部署

### Hextra 主题特色

[Hextra](https://github.com/imfing/hextra) 是一个现代化的 Hugo 主题：

1. **现代化设计** - 简洁、专业
2. **全文搜索** - 内置 Fuse.js 搜索
3. **响应式布局** - 完美适配移动端
4. **暗黑模式** - 自动/手动切换
5. **多语言支持** - i18n ready

## 代码示例

Hextra 支持语法高亮：

```python
# Python 示例
def hello_world():
    print("Hello, Hugo!")
    
hello_world()
```

```javascript
// JavaScript 示例
const greeting = () => {
    console.log('Hello, Hextra!');
};

greeting();
```
